【各種機能】
　・指定したホットキーで瞬時に呼び出し可能
　・画像を添付可能
　・クリップボードの画像を直接アップロード可能
　・スクリーンショットを撮影してそのままアップロード可能
　・規制までのおおよその目安を表示
　・規制時にアカウントを自動で切り替える機能を実装
　・ツイート時のハッシュタグ自動付加にも対応
　・添付した画像を確認することが可能に

【使用方法】
　1. TwiField.exeを実行します。
　2. ダイアログ右上の「アカウント追加」ボタンを選択します。
　3. アカウント認証をします。
　4. その他設定を変更して「保存」ボタンを選択します。
　5. ホットキー(初期値: Ctrl + Space)の同時押し または
　   タスクバーのアイコンをクリックします。
　6. 呟く内容を入力します。
　7. Enterキー（初期設定）で投稿します。

【既知のバグなど】
　・残規制数の表示が-1になったときは，残規制数の取得に必要なTLの取得に失敗
　　しています．規制数の使用をOFFにするか，アカウントあるいはviaを切り替え
　　て下さい．
