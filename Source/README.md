TweetField
==========
This application provides you with easy-to-use abundant functions.

1. Function
----------
+ Correspond HotKey
+ Post Tweet with Picture
+ Post Tweet with Picture on Clipboard
+ Post Tweet with ScreenShot
+ Automatic Addition about a Hash Tag
